About SkillSwap Hub

SkillSwap Hub is a full stack web application designed to support collaborative learning through a community driven platform. Users can share knowledge, discover new skills, and engage with others through interactive features such as posting resources, commenting, and upvoting.

Our Mission

The mission of SkillSwap Hub is to make learning accessible, inclusive, and engaging for individuals from all backgrounds. By combining modern web technologies with community interaction, the platform encourages knowledge exchange and skill development.

Key Features

  • User authentication system (login & registration)
  • Profile creation and personalisation
  • Resource sharing with categories
  • Search and filtering functionality
  • Upvoting and commenting system
  • REST API integration with Flask backend

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Flask (Python)
  • Database: MongoDB
  • Security: Flask-Bcrypt for password hashing
  • Architecture: Client server model with REST API

Academic Context

This project was developed as part of the COM4113 module and demonstrates full stack development skills. It includes both frontend interface design and backend implementation, with a focus on usability, responsiveness, and secure data handling.
